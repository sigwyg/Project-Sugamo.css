E[att^=val]

att属性の値が文字列valで始まるE要素。
属性値の文字列から、簡単なパターンマッチングを可能とする。

<li class="sample1">サンプル１</li>
<li class="nosample">サンプルではない</li>
<li class="sample2">サンプル２</li>
  • サンプル１
  • サンプルではない
  • サンプル２
#selectors2 .sample1 li[class^="sample"] {
    background: #f99;
}

E[att$=val]

att属性の値が文字列valで終わるE要素。
属性値の文字列から、簡単なパターンマッチングを可能とする。

<a href="sample.pdf">PDFファイルへのリンク</a>
  • PDFファイルへのリンク
#selectors2 .sample2 a[href$=".pdf"] {
    padding-right: 20px;
    background: transparent url(./img/icon_pdf.gif) no-repeat center right;
}

E[att*=val]

att属性の値に文字列valを含むE要素。
属性値の文字列から、簡単なパターンマッチングを可能とする。

<a href="/web/html5-demo.html">HTML5</a>
<a href="/travel/maui.html">Travelling Maui</a>
<a href="/web/css3-demo.html">CSS3-demo</a>
  • HTML5
  • Travelling Maui
  • CSS3
#selectors2 .sample3 a[href*="demo"]:after {
    content: " !";
    color: red;
    font-weight: bold;
}

E:target

URIに含まれるフラグメント識別子によって特定されたE要素。

フラグメント識別子はURIの「#」以下の部分。「http://www.example.com/mypage.html#section_3」でいうところの「section_3」。つまり、「section_3」という名前(id)で特定されるE要素が、目的地アンカーとして参照された時に適用される。

<article id="selectors2">
    <h1>selectors (2/2)</h1>
    ... goodness ...
</article>

#samples article {
    width: 0;
    height: 0;
    overflow: hidden;
}
#samples article:target {
    width: 500px;
    height: auto;
}

E:indeterminate

オンでもオフでもない未確定な状態であるUI要素、としてのE要素。
現在のCSS3仕様では定義されていない(削除された)が、将来の仕様では紹介するとされる。

通常、ラジオボタンとチェックボックスはユーザーの手によってチェックされたり、チェックを外されたりする。だが時にそのどちらでもない状態になることがあって、これは要素の属性値として指定されたり、DOM操作によってそういった状態をとる。

<input type="radio" name="sample3" id="idm" /> <span>dog</span>
<input type="radio" name="sample3" /> <span>cat</span>
<input type="radio" name="sample3" /> <span>hedgehog</span>
<script>
    var idm = document.getElementById('idm');
    idm.indeterminate = true;
</script>
  • dog
  • cat
  • hedgehog
#selectors2 .sample7 input:indeterminate + span {
    background: #f99;
}

  • hedgehog

E::selection

E要素の選択領域。現在は、CSS3仕様から外されています。

ここは<p#selection-sample>です。この要素内で選択した部分は背景が赤になります。

#selection-sample {
    border: 1px solid #000;
    padding: 0.5em;
    margin: 0 3em 1.5em;
}
#selection-sample::selection {
    background: #f99;
}
